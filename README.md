# My Discord Bot

A Discord bot built using [discord.js](https://discord.js.org).

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/my-discord-bot.git
   ```
2. Edit config.json accordingly
3. Run the command:
  ```bash
  cd "{path to your bot folder}"
  ```
4. Final command:
  ```bash
  node .
        OR
  node index.js
  ```
